画像編集ツールです。

指定したフォルダ内の画像のサイズを変更します。
また連番のファイル名で新しいファイルを作成します。

ブラウザで動作するので、Google Chrome、InternetExplorer、Safariなどで、editPhotoFiles.htmlを開いてください。

※ブラウザが古い場合は動作しない可能性があります。動作しない場合は、「ブラウザが非対応です」というポップアップが表示されます。


〜事例ページ更新時の使い方〜

・外部ライブラリを使用しているので、ネットに繋がる環境で起動してください。
・事前に、編集対象のファイルを一つのフォルダにまとめておいてください。
・編集後のファイルの保存先は、ブラウザに設定しているダウンロードファイル保存用のパスになります。
・事例更新時は、長編サイズ、画質共にデフォルトのもので問題ありません。
